PROJECT OVERVIEW

A2RL Autonomous Drone Racing

The A2RL Autonomous Drone Championship is a multi-year drone racing competition hosted in Abu Dhabi. This project is part of our preparation process for the third competition year (2026-2027), developing vision-based autonomous navigation using ArUco markers and ROS2.

SYSTEM ARCHITECTURE

Sensing - ArUco Marker Tracking

Four-stage pipeline for autonomous drone control through visual tracking

1
Align with Marker
    Camera Detection
  • Monocular Camera (Arducam)
  • Low latency processing
  • Real-time marker detection
  • Align with marker center
2
Publish Coordinates
    ArUco Node
  • Publish marker pixel coords
  • Subscribe: /arducam/image_raw
  • Process: cv2.aruco
  • Publish: /marker_pixel
3
Compute Error
    Error Calculation
  • Calculate x/y pixel error
  • error = marker ‚àí image_center
  • Feed into planning node
  • Real-time error tracking
4
Velocity Output
    Control Command
  • Planner outputs velocities
  • Correct positioning error
  • Smooth trajectory
  • Publish: /desired_pose
