PROJECT OVERVIEW

A2RL Autonomous Drone Racing

The A2RL Autonomous Drone Championship is a multi-year drone racing competition hosted in Abu Dhabi. This project is part of our preparation process for the third competition year (2026-2027), developing vision-based autonomous navigation using ArUco markers and ROS2.
